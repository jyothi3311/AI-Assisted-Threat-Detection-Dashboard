# AI-Assisted-Threat-Detection-Dashboard

## Overview
This project is developed as part of the Infosys Springboard Internship.

The system uses Artificial Intelligence and Machine Learning to detect cybersecurity threats, classify attacks, and visualize results through an interactive dashboard.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Plotly
- Google Cloud Platform (GCP)
